Project: Airbnb Clone

I developed a full-stack Airbnb clone with a responsive user interface and dynamic backend. This clone allows users to browse, book, and list properties just like the real platform. Key features include:

  • User authentication and secure login with JWT
  • Property listing with images and detailed descriptions
  • Search functionality based on location, price, and availability
  • Booking system with real-time calendar updates
  • Backend developed using Node.js and Express
  • Database management using MongoDB

The project is built with React on the frontend, making use of reusable components and state management with Redux. The backend is powered by Node.js and Express, while the database uses MongoDB. It mimics the functionality of Airbnb, offering a seamless user experience.
